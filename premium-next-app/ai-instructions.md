# TechBridge Website Project Rules

## Identity & Purpose
TechBridge is a premium software development and AI automation agency. The website must communicate technical expertise, modern engineering capability, high-end digital craftsmanship, and trust with B2B clients. 

## Target Audience
Small to medium businesses, startups, and organizations that need modern software systems and AI automation.

## Core Services
- Custom software development
- AI workflow automation
- Internal business tools
- SaaS platforms
- API integrations
- Data dashboards

## Tech Stack
- Framework: Next.js (App Router)
- Language: TypeScript
- Styling: Tailwind CSS
- Animation: Framer Motion & GSAP (ScrollTrigger)
- Smooth Scroll: Lenis
- 3D Rendering: Three.js / React Three Fiber (R3F)

## Design Direction
- Aesthetic: Dark luxury tech, "Liquid Glass" patterns
- Colors: Black / charcoal background with subtle gradients
- Typography: Clean, modern, highly readable
- Layout: Minimal but powerful UI, premium spacing, Bento Grid where applicable
- Motion: Smooth, highly optimized animations (prevent layout thrashing)

## Development Rules
- Architecture: Clean, modular, reusable components.
- 3D Architecture: Isolate 3D in its own directory. The primary visualization is a central "Neural Bridge". Load canvas dynamically for fast initial page load times.
- Execution: Focus on UI and layout first. Do not build complex 3D scenes without explicit approval.
